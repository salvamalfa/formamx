## Contexto

Migración de una tienda Shopify tradicional a arquitectura headless para mejorar rendimiento y libertad de diseño.

## Implementación

- Frontend en Next.js 14 con App Router
- Datos via Shopify Storefront API (GraphQL)
- ISR (Incremental Static Regeneration) para catálogo
- Checkout nativo de Shopify para pagos seguros

## Resultados

- Score Lighthouse: 98/100
- Tiempo de carga < 1.2 segundos
- Tasa de conversión mejoró 22%
