## El problema

El cliente necesitaba centralizar métricas de ventas, soporte y marketing en un solo lugar, con actualización en tiempo real.

## Solución

Desarrollé un dashboard modular con React y TypeScript. Los datos se obtienen via WebSocket desde una API REST en Node.js. Para las gráficas usé D3.js con transiciones fluidas y soporte para modo oscuro.

## Resultado

- Reducción del 40% en tiempo de reporte semanal
- Integración con 3 fuentes de datos externas
- Exportación a PDF/CSV con un clic
